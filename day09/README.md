## 测试

在 `*_test.go`文件中，有三种类型的函数：单元测试函数、基准测试函数和示例函数。

| 类型         | 格式           | 作用                           |
| ------------ | -------------- | ------------------------------ |
| 单元测试函数 | `TestXXX`      | 测试程序的一些逻辑行为是否正确 |
| 基准测试函数 | `BenchmarkXXX` | 测试性能                       |
| 示例函数     | `ExampleXXX`   | 为文档提供示例                 |
